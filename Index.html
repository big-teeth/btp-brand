Critical Color Rules

  • ✕ Never use #444, #555, #777, #888, #111, #222 — not in the palette. Map to nearest palette color.
  • ✕ Never use small red text on dark backgrounds. Red on dark only works at display/headline sizes (48px+).
  • ✕ Red backgrounds (#B20022) are for the CTA section and single-line statement bars only. Use #333333 for callouts and handoffs.

Guardrails Non-negotiable rules for every build

Never Do — Design

  • ✕ Rounded corners on anything (border-radius: 0)
  • ✕ Horizontal red rules under section titles
  • ✕ Left red accent bars on dark sections
  • ✕ Small red text on dark backgrounds
  • ✕ Red backgrounds except for CTA + statement bars
  • ✕ Colors outside the 7-color palette
  • ✕ Barlow Condensed for body copy
  • ✕ Barlow Regular for body copy (use Medium 500)

Never Do — Wix Studio

  • ✕ Scale Proportionally on text elements (use Wrap)
  • ✕ Skip Apply Max Width on any section
  • ✕ Set text opacity directly (use a darker hex instead)
  • ✕ Fixed heights on content containers
  • ✕ Percentage-based padding (use px only)
  • ✕ Galleries for precise layout (use CSS containers)
  • ✕ Hero section height: Scale Proportionally (use Fit to Screen)
  • ✕ Leave image alt text blank

Deviations from the PDF Brand Guide — Web versions are correct

Body Copy Weight

PDF says Regular (400). Web uses Medium (500). Always use Medium on web.

Red Rules & Accent Bars

PDF shows horizontal red rules under titles and left accent bars. Both removed from web permanently.

Special Elite Font

PDF references Nitti Typewriter. Web uses Special Elite. Same role, different font. Special Elite is correct for web.
